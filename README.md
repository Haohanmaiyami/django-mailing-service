# Sending Messages Service app

Веб-приложение на Django для управления email-рассылками.

##  Функциональность

- Регистрация и авторизация пользователей
- Создание клиентов, сообщений и рассылок
- Ограничение доступа: только свои объекты
- Отправка рассылок вручную и через консольную команду
- Ведение логов попыток отправки
- Кэширование статистики на главной странице

##  Отправка рассылок

- Ручная отправка возможна из интерфейса "Список рассылок"
  (python manage.py send_mailings)

## ✉️ Отправка рассылок

- Ручная отправка возможна из интерфейса **"Список рассылок"**.
- Также можно вызвать через консольную команду:
```
python manage.py send_mailings
```
